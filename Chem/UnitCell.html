    class Chem::UnitCell

    • Chem::UnitCell
    • Reference
    • Object

    Overview

    A parallel epipedal unit cell defined by three vectors (basis).

    It is internally represented by a 3x3 matrix, where each column correspond to a basis vector (see Spatial::Mat3.basis), which coordinates are expressed in Cartesian space (angstroms). In this way, the basis matrix can be used to transform from Cartesian to fractional coordinates, and viceversa, by matrix multiplication (see #cart and #fract).
