    enum Chem::PDB::Writer::BondOptions

    Overview

    Controls which bonds are written to a PDB.

    Defined in:

    chem/formats/pdb.cr

    Enum Members

    Standard = 1

    Write CONECT records for standard residues (including water)

    Disulfide = 2

    Write CONECT records for disulfide bridges

    Het = 4

    Write CONECT records for non-standard (HET) groups excluding water (both intra- and inter-residue bonds)
